Welcome to the first post on imick.io. This site is a long-term home for my writing,
a place to publish code snippets I keep reaching for, and a portfolio that doesn't
disappear into a feed two days after it ships.

## Why now

Every piece of content I've posted to X, LinkedIn, or YouTube has had no durable
home. A personal domain compounds — search rankings, brand equity, and the simple
fact of *being able to point at one URL* all add up over time.

## What to expect

Roughly one article a week. A mix of:

- Opinions about how to build software
- Technical deep-dives, usually grounded in code I'm shipping
- Short notes on tools and patterns

## The stack

This site is built on Next.js with the App Router, MDX for content, and Tailwind
for styling. Here's the snippet that detects whether a post is a draft:

```ts title="lib/posts.ts"
export function isDraft(post: Post): boolean {
  if (!post.publishedAt) return true
  const published = new Date(post.publishedAt).getTime()
  if (Number.isNaN(published)) return true
  return published > Date.now()
}
```

A post with no `publishedAt`, or one set in the future, is treated as a draft —
hidden in production, visible in `next dev` with a `DRAFT` badge.

## Open source

The whole site is open source. If you spot a typo or want to see how something is
wired up, the repo is linked from the footer. More on that in a future post.
